## Description
Build an accessible, React-based finance dashboard that visualizes personal spending data. The project focuses on semantic HTML, keyboard navigation, ARIA usage, and screen reader-friendly data visualizations while maintaining a clean, modern UI suitable for a junior front-end role.

## Tech Stack
- React

## Requirements
- React-based finance dashboard
- Semantic HTML
- Keyboard navigation
- ARIA usage
- Screen reader-friendly data visualizations
- Clean, modern UI

1. Scaffold React app structure.
